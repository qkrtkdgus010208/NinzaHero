## 👨‍🏫 프로젝트 소개

2019년 5월 16일 Habby[1]에서 출시한 모바일 게임, 궁수의 전설을 레퍼런스로 만든 로그라이크 형식을 사용한 탑다운 게임 "NinjaHero" 을 1조에서 제작했습니다.

## 팀소개
저희 팀 1조는 궁수의 전설을 레퍼런스로 삼아서 유니티에서 2d core를 이용해 탑다운 게임을 제작하기로 했으며 각자가 게임의 한 부분을 맡아서 게임을 제작했습니다.
### 팀의 역할 분담
박상현: 전투시스템, 애니메이션 구성, 스킬 제작

송덕희: UI

정지원: 맵 구성, PPT제작, 시연영상 제작

오민근: 보스 몬스터

송정민: 일반 몬스터, 스테이지 관리, 발표



## 프로젝트 계기
입문 프로젝트 발제 후, 3가지 게임 중에서 궁수의 전설이 제일 난이도가 높고 만들면서 배울게 많을 것 같다는 생각이 들어 팀에서 궁수의 전설을 레퍼런스로 게임을 만들기로 하게 되었습니다.

## 💜 주요기능

- 기능 1
캐릭터가 일정범위에 있는 몬스터를 타겟으로 지정 한 뒤 자동공격
- 기능 2
스테이지 마다 랜덤으로 나오는 몬스터
- 기능 3
스테이지 시작시 스킬을 랜덤으로 고를수있는 슬롯모션
- 기능 4
보스 몬스터의 공격 패턴과 즉사 스킬

```
*보스 공략법
보스맵에는 보스 말고 다른 몬스터들도 함께 나오는데 그때 다른몬스터를 먼저 타겟팅해서 플레이어가 보스몬스터를 먼저공격하지 않을 수 있기 때문에 다른 몬스터들을 먼저 처지한 뒤에 보스 몬스터를 공격하는게 좋다
```
